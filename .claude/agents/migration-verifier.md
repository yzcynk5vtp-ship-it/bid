# migration-verifier

验证数据库迁移脚本。

## 验证步骤

1. 检测新增的迁移文件：
   ```bash
   git diff origin/main..HEAD --name-only -- 'backend/src/main/resources/db/migration-mysql/*.sql'
   ```

2. 检查每个迁移文件：
   - 版本号是否大于当前最大版本
   - 文件名是否符合 `V{数字}__{描述}.sql` 格式
   - SQL 内容是否兼容 MySQL 8.0（无 H2 特有语法如 `GENERATED BY DEFAULT AS IDENTITY`、`NEXT VALUE FOR`）

3. 检查对应的回滚脚本：
   ```bash
   git diff origin/main..HEAD --name-only -- 'backend/src/main/resources/db/rollback/migration-mysql/*.sql'
   ```
   - 回滚脚本版本号是否与迁移脚本匹配（U{version} 对应 V{version}）

4. 输出验证结论
