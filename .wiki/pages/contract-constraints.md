# 合同约束

> 本页是“西域数智化投标管理平台建设项目”的合同硬约束索引。所有研发、实施、验收、付款和变更判断，优先回到本页及其原始合同源文件核对。
> 当前开发、产品规划和实施主执行基准参见 [[implementation/sow-2026-v1-4]]。
> 附件4需求任务书的章节级交付核查参见 [[implementation/attachment4-requirement-task-book]]。

## 1. 约束优先级

| 层级 | 执行口径 |
|---|---|
| 主合同 | 合同正文优先；附件与正文不一致时，以主合同为准 |
| SOW V1.4 / 需求任务书 | SOW V1.4 是当前项目实施、开发排期、产品规划、上线切换和验收的主执行基准；作为服务内容、实施范围、文档交付、验收准则的直接依据 |
| 报价清单 | 作为固定总价、模块报价、人天、运维费和增补报价的依据 |
| 双方书面确认件 | 蓝图、接口清单、报表定义、阶段确认、验收报告等一经签发即成为执行基线 |

执行提醒：任何新增需求、范围变更、计划调整、接口清单变化、报表口径变化，都必须走双方书面确认或补充协议。

## 2. 合同主体与许可

| 项 | 约束 |
|---|---|
| 项目名称 | 西域数智化投标管理平台建设项目 |
| 甲方 | 西域智慧供应链（上海）股份公司 |
| 乙方 | 上海泊冉软件有限公司 |
| 软件产品 | 泊冉客户管理系统 V1.0 |
| 授权范围 | 甲方及甲方关联方获得不可转让、非独占的永久使用权 |
| 知识产权 | 标准软件著作权归乙方；系统二次开发部分、甲方业务资料和系统数据归甲方 |
| 用户许可 | 合同正文写明授权模块及 200 模块用户许可证数量；报价单同时出现“包含 200 注册用户”和“使用人数 100 人、增购账号 100 元/个”的口径，执行前需按最终盖章文本确认 |

许可触发条件：配置实施、安装调试、系统测试阶段完成并满足约定功能需求，相关阶段文档经甲方确认且双方共同签发后，软件许可使用权正式授予甲方及关联方。

## 3. 固定价与费用边界

| 费用项 | 金额 |
|---|---:|
| 合同总价 | 350000 元 |
| 软件产品授权许可费 | 90000 元 |
| 实施服务费 | 180000 元 |
| 集成服务费 | 80000 元 |
| 首年运维费 | 0 元 |
| 后续运维服务费 | 30000 元/年 |
| 履约保证金 | 20000 元 |

固定价边界：

- 合同总价为固定价，覆盖 SOW 范围内的软件许可、实施、定制开发、办公、交通、人员、餐旅、文档、税费等费用。
- 报价清单中约定的人天数量是完成任务所需全部人天，乙方不得因实际耗用超出报价人天而追加合同价款。
- 第三方 API、账号、授权、测试环境、网络白名单由甲方或甲方协调第三方提供；第三方接口费、账号费、订阅费、AI Token 费不含在本期实施报价内。
- 财务系统等合同范围外新增需求，应按报价清单中实际人天价格另行核算并签补充协议。

## 4. 付款节点

| 节点 | 触发条件 | 比例 | 金额 | 支付方式 |
|---|---|---:|---:|---|
| 1 | 合同生效 7 日内，乙方人员配置到位并经甲方审查通过，整体工作计划及人员分工经甲方书面确认 | 10% | 35000 元 | 90 日银行承兑汇票 |
| 2 | 需求适配阶段全部文档资料经甲方确认并由双方共同签发 | 10% | 35000 元 | 120 日供应链金融 |
| 3 | 方案设计阶段全部文档资料经甲方确认并由双方共同签发 | 10% | 35000 元 | 120 日供应链金融 |
| 4 | 配置实施、安装调试、系统测试阶段完成，软件系统测试上线，三阶段文档经甲方确认并由双方共同签发 | 40% | 140000 元 | 120 日供应链金融 |

付款暂停规则：乙方在付款前存在违反合同约定行为时，甲方有权暂停该笔及后续款项，直至双方就违约处理达成一致。

## 7. 团队与驻场

| 约束 | 要求 |
|---|---|
| 团队构成 | 至少包含项目经理、开发工程师、测试工程师、实施工程师 4 类角色 |
| 项目经理 | 5 年以上投标管理系统项目管理经验；不得由项目总监兼任；需求调研和方案设计阶段全程现场出席 |
| 方案负责人 | 配置实施阶段每周现场至少 2 天 |
| 实施顾问 | 必须有专门顾问负责需求适配，每个模块需有骨干顾问支撑 |
| 人员稳定 | 人员经甲方面试合格进入项目组后原则上不得变更；离职等极特殊变更需甲方审查通过并现场交接 |

人员配置达不到附件 4 要求时，甲方可作为中标否定条款；合同执行中人员配置未达标也会影响首付款触发。

## 9. 验收与质量门禁

| 验收类别 | 准则 |
|---|---|
| 软件平台验收 | 软件平台及模块满足全部功能需求，符合数字化建设要求、信息化总体规划、信息安全等级保护要求，并具备良好接口扩展性 |
| 文档资料验收 | 纸质与电子文档满足需求任务书要求，重要文件通过甲方组织评审 |
| 培训验收 | 培训计划和验收标准经双方确认，甲方人员理解系统功能并基本掌握操作，完成知识转移 |
| 实施过程验收 | 通过甲方组织的全部质量控制点评审 |
| 系统整体验收 | 经过试运行验证，系统符合需求任务书全部要求并达到建设目标 |

质量控制点包括蓝图/配置/实施/安装调试/系统测试/培训推广/上线准备/运行/验收等阶段。质量控制点评审通过并由甲方项目经理及质量控制小组联合签字后，方为通过。

| 运维续费 | 免费期满后双方另签运维协议，标准 30000 元/年 |

## 11. 违约与合规红线

| 事项 | 合同后果 |
|---|---|
| 任一阶段或时间节点延迟 | 视为重大违反合同；甲方可要求每日按合同总价万分之五支付违约金，或解除合同并要求退款及 20% 违约金 |
| 实施团队人员违规变更 | 按实施服务费用 5% 支付违约金 |
| 其他违约行为 | 甲方可解除合同，要求退还已付款并支付合同总价 20% 违约金 |
| 知识产权承诺违约 | 甲方可解除合同，要求退还已付款并支付合同总价 30% 违约金，不足部分继续赔偿 |
| 保密违约 | 支付主合同金额 30% 违约金，不足部分继续赔偿 |
| 商业贿赂 | 甲方可解除合同并要求主合同金额 30% 违约金；供应商反行贿准则另约定罚金按更高者执行 |
| 履约保证金不足 20000 元 | 乙方须立即补足；迟延补足按迟延金额每日千分之一支付违约金 |
