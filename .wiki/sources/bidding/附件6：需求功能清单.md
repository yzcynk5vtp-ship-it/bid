## 工作表1
| 一级模块 | 二级模块 | 核心功能点 | 功能描述与核心要求 |
| --- | --- | --- | --- |
| 项目全流程管理 | 统一项目工作台 | 投标数据及待办 | 为销售、投标团队、管理层提供角色化视图，集中展示个人待办任务、负责的项目及全局投标日历。 |
| | | 一站式流程发起 | 支持销售快速发起：标书支持申请、资质/合同借阅申请、投标费用申请（保证金/标书购买费）。 |
| | | 智能日程与预警 | 集成获取关键时间节点（如截标日、开标日、获取合同合作期限计算续标日），提供系统消息、邮件等多级超前预警（按照客户、项目进行不同时间维度的提醒配置），确保关键节点零遗漏。 |
| | 标讯信息管理 | 外部标讯获取与入库 | 多渠道接入：支持集成外部招标平台数据或接入第三方商机服务，第三方商机服务支持配置关键字，及时获取与公司业务相关的招标公告信息，支持人工录入标讯信息。\n 建立企业标讯库：所有获取的招标信息统一进入标讯库，支持按时间、地区、采购方、关键词等进行检索与筛选。 |
| | | 标讯分发与指派跟进 | 智能/手动分发：支持根据预设规则（如区域、产品线）自动将标讯指派给相应销售负责人，也支持管理员手动分发。\n 跟进状态管理：销售领取标讯后，需在系统中更新状态（如“已联系”、“放弃”、“参与投标”等），形成从信息到项目的跟进闭环。 |
| | | 超前预测与市场洞察 | 趋势预测：基于历史标讯/合同数据，提供简单的超前预测分析，如热点采购行业趋势、特定采购方招标规律等。\n机会预判：结合预测分析辅助识别高潜力市场机会。 |
| | 投标项目过程管理 | 项目立项与信息登记 | 支持从CRM集成客户信息，创建投标项目，记录客户、招标平台、时间、标讯、竞争对手等核心信息。\n项目信息可在项目全流程的过程中及时更新。 |
| | | 流程驱动与任务协同 | 支持可配置的审批流程（如立项评审）和流程字段要求，并能将标书编制任务在线分解、分配给技术、商务等成员。 |
| | | 执行跟踪与交付物关联 | 跟踪拆解任务进度状态、获取交付物信息，形成项目任务闭环，提交至标书编写流程 |
| | | 标书编制发起与核查 | 支持标书初稿、内部评审等过程管理，获取用印、封装提交等各环节状态，并关联最终的标书文件作为交付物。 |
| | | 投标结果闭环 | 项目结束后，必须人工登记中标/未中标结果以及合同合作期限（支持从内部系统获取），中标支持填写备注信息记录（如中标SKU数量）以便后续使用分析。\n提醒销售上传中标通知书或分析报告，完成知识沉淀与流程闭环。\n支持自动抓取公开信息的中标结果，提醒销售人工确认。\n已中标的竞争对手记录，包含SKU数量、品类、折扣、账期等信息，形成分析报表。 |
| 企业知识资产中心 | 统一资质与案例库 | 商务资质库 | 集中管理公司及子公司各类证照、产品资质、人员证书，支持有效期管理、提醒以及借阅。 |
| | | 历史项目与案例库 | 结构化归档历史投标的技术/商务标书，支持按行业、产品、结果等标签分类与全文检索，形成可复用的案例资产。 |
| | 投标模板与知识库 | 标准模板库 | 建立按产品类型、行业、文档类型（技术方案、商务应答等）分类的标书模板库。 |
| 资源管理 | 投标费用管控 | 费用申请与支付跟踪 | 线上化完成标书购买费、投标保证金的申请、审批与支付记录跟踪流程。相关费用（例如，投标差旅费用、标书制作费等）归纳汇总于具体投标项目中。 |
| | | 保证金归还提醒 | 根据开标结果以及退款时间，系统自动触发对保证金退还情况的跟踪提醒。 |
| | | 费用台账与统计 | 归集所有投标相关费用，形成台账，支持按项目、时间、部门等多维度查询与统计。 |
| | 外部平台账户管理 | 平台账户集中管理 | 建立公司各类外部招标平台账户的统一登记簿（如政府采购网）。 |
| | | 账户借阅管理 | 销售投标时需在线申请借阅账户，系统记录借阅用途（购标书、投标）并关联项目，形成完整、可审计的使用轨迹。 |
| 数据分析 | 投标数据看板 | 管理层可视化仪表盘 | 动态展示年度/季度投标数量、中标率、中标金额趋势等核心经营指标。 |
| | | 多维深度分析报表 | 支持按区域、产品线、销售团队、客户类型、竞争对手等维度进行中标分析、投入产出分析。 |
| | | 数据穿透下钻 | 在看板与报表中，可点击数据下钻查看具体项目详情、完整过程文件及团队信息。 |
| 权限管理 | 权限管理 | 权限配置与管理 | 支持系统按照组织架构（从项目组、用户等维度）配置系统/数据权限 |
| 系统集成与扩展 | 核心业务集成 | 与CRM系统集成 | 支持从CRM获取客户信息、商机信息等 |
| | | 与组织架构系统集成 | 支持获取西域内部组织架构信息等 |
| | | 与OA/审批流集成 | 实现组织架构用户同步、单点登录，并获取用印、合同评审、付款申请等专用的OA审批流程。 |
| | | 开放API接口 | 提供标准API，与财务系统、电子签章等集成预留能力，保障系统扩展性。 |
| | 智能辅助 (AI扩展) | 标书合规性自动检查 | 基于预设规则库（如必备资质条款、禁止性内容），自动对标书文档进行符合性初检并提示风险，模拟专家评审评分 |
| | | 文本书写质量辅助 | 提供错别字、语法、格式规范性的智能检查与修改建议。 |
