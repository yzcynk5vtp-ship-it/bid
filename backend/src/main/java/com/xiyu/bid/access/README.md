# Access 模块 (访问控制纯策略)

> 一旦我所属的文件夹有所变化，请更新我。

## 职责
提供跨业务模块复用的访问控制纯核心策略。这里不读取数据库、不读取安全上下文、不写日志，只根据调用方传入的权限快照返回决策。

## 边界清单
| 文件 | 地位 | 功能 |
|------|------|------|
| `core/ProjectLinkedRecordVisibilityPolicy.java` | Pure Core | 判断项目关联记录是否对当前用户可见 |

## 约束
- 当前用户、管理员身份、可见项目集合必须由应用服务或权限服务传入。
- 不在本模块引入 Repository、Controller、Spring Security 或其他副作用依赖。
