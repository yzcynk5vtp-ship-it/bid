# performance

历史中标/合同台账，供标书编制时一键引用，支持相似业绩推荐。

## 已实现功能

### 核心数据管理
- 业绩列表查询（多维度筛选：关键词、客户类型、项目类型、状态、客户级别、属地、日期范围、中标通知书、项目负责人）
- 业绩新增 / 编辑 / 删除
- 详情抽屉（5 Tab：合同基础 / 关键日期 / 客户信息 / 附件资料 / 操作日志）

### 业务校验
- 截止日期 > 签约日期
- 总截止日期 ≥ 截止日期
- 联系方式格式（手机/固话/邮箱）
- 央企客户必填附件（央企名录截图 + 关系证明）
- 中标通知书联动校验
- 敏感字段修改警示弹窗（客户类型 / 签约日期）

### 附件管理
- 7 种附件类型（合同协议、商城截图、央企名录、品类页、关系证明、中标通知书、其他）

### 导入导出
- Excel 模板下载
- Excel 批量导入（含校验报告）
- Excel 导出
- ZIP 导出（含 Excel 台账 + 附件文件）

### 到期提醒
- 合同状态自动计算（履约中 / 即将到期 / 已到期）
- 差异化提醒窗口（央企 180 天 / 其他 90 天）
- 定时扫描任务（每天 9:00）
- 提醒配置（可配置天数 + 开关，仅 ADMIN）

### 操作审计
- 所有操作记录审计日志（@Auditable）
- 时间轴展示

### 权限控制
- ADMIN / MANAGER / STAFF 角色分级
- 创建/编辑/删除：ADMIN + MANAGER
- 查看：ADMIN + MANAGER + STAFF
- 导出：ADMIN + MANAGER
- 提醒配置：仅 ADMIN

### 相似业绩推荐（标书编制集成）
- 复用四库联动匹配结果（`KnowledgeBaseMatchAppService`）
- 标书编制页面「推荐业绩」按钮
- 展示匹配业绩列表（已满足 / 需关注 / 不满足）
- 支持复制业绩信息到剪贴板

## 文件清单

| 文件 | 功能 |
|------|------|
| `domain/` | 业绩记录值对象、业绩匹配策略、仓储接口 |
| `application/` | 创建、更新、删除、列表、导入导出、到期提醒扫描等用例编排 |
| `infrastructure/` | JPA 实体、仓储适配、REST 控制器 |
