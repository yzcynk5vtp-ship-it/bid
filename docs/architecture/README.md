# 架构文档 (architecture/)

存放架构设计、安全设计、测试策略等系统级文档。

## 目录职责

本目录存放技术架构设计文档、模块实现计划、安全设计、测试策略等系统级文档。这些文档描述系统的技术实现方案和架构决策。
