2. 六阶段说明

立项 → 标书制作 → 评标中 → 结果确认 → 项目复盘 → 项目结项，共 6 个阶段。

1

项目立项

两个入口：标讯评估通过 / CRM 一键发起。建档、分配负责人、初步预算。

2

标书制作

技术 / 商务 / 资质三方协作，调用知识库 + AI 辅助起草，多轮评审。

3

评标中

跟进开标进度，记录答疑澄清、专家评审反馈，等待评标结果公示。

4

结果确认

收到中标/落标通知，登记结果、上传中标通知书、回写 CRM 商机状态。

5

项目复盘

中标 / 落标皆复盘：成败要素、报价合理性、流程改进点沉淀。

6

项目结项

归档全部材料、保证金 / 履约金回收、知识资产入库。
