# 共享资源 (assets/)

存放图片、设计稿、共享资源等文件。

## 目录职责

本目录存放各类共享资源文件，包括图片、Excel 表格、设计稿等二进制资源。这些文件被多个文档或模块引用。

## 文件清单

| 文件 | 类型 | 功能 |
|------|------|------|
| `启动前项目准备进度计划表-2026-04-27.xlsx` | Excel | 项目准备进度计划表（Excel 版） |
| `启动前项目准备进度计划表-2026-04-27.md` | Markdown | 项目准备进度计划表（Markdown 版） |
| `favicon.ico` | 图标 | 网站 Favicon |

## 更新规则

- 新增图片资源时请使用有意义的文件名
- 定期清理不再使用的资源文件
- 大文件（> 10MB）建议使用外部存储引用
