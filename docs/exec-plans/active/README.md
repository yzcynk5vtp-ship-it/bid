| 文件 | 功能 |
|---|---|
| (空) | 当前无正在执行的复杂任务计划 |

## 约定

收到包含 `Phase`、`开发计划`、`需求开发` 等复杂任务时：
1. 优先按 `PLANS.md §Spec Kit 流程门禁` 走 speckit 流程
2. 在本目录补一份 `<task-slug>-plan.md`（目标 + 进度清单 + 决策日志）
3. 任务完成后移入 `../completed/`

> 既有活跃计划仍在 `docs/plans/`，本目录为新增复杂任务落点。
