# BAR 投标资产台账模块设计文档

## 概述

**模块名称**：BAR (Bidding Asset Registry) 投标资产台账 + 流程管控

**核心目标**：让任何一个投标动作开始前，能立刻知道：
- 这个站点有没有账号
- 账号归属
- 能不能登录
- 需要哪个UK
- UK在谁手里
- 找回路径是什么

---

## 1. 整体架构

```
资源管理
├── 费用管理（已有）
├── 账户管理 → 升级为"资产台账BAR"
└── ┌─ "可投标能力检查"（默认首页）
    └─ "站点台账"（Tab切换）
```

---

## 2. 核心页面结构

### 2.1 可投标能力检查面板（独立入口）

**输入**：站点名称/网址搜索框

**输出**：简洁卡片式结果
```
┌─────────────────────────────────┐
│ 🔍 中国政府采购网                │
├─────────────────────────────────┤
│ ✅ 账号状态：已注册              │
│ 📱 登录方式：密码 + CA           │
│ 🔑 需要UK：法人CA（在库）         │
│ 👤 责任人：张三 138****1234      │
│ ⚠️  风险：CA即将过期(30天)        │
│                                 │
│ [查看详情] [申请借用]            │
└─────────────────────────────────┘
```

**状态定义**：
- 🟢 可投标：账号可用 + UK在库 + 无风险
- 🟡 有风险：账号可用但UK借出/即将过期
- 🔴 不可投标：无账号/账号异常/UK遗失

│                                                              │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ 平台名称        │ 登录方式  │ 账号 │ UK  │ 状态  │ 操作 ││
│  ├─────────────────────────────────────────────────────────┤│
│  │ 中国政府采购网    │ 密码+CA   │ 2   │ 2   │ 🟢    │ 详情 ││
│  │   ccgp.gov.cn   │           │     │     │       │ 编辑 ││
│  ├─────────────────────────────────────────────────────────┤│
│  │ 广东省公共资源   │ 仅CA     │ 0   │ 1   │ 🟡    │ 详情 ││
│  │   gdggzy.cn     │           │     │     │ 风险  │ 编辑 ││
│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
```

**列表字段定义**：
- **平台名称**：显示站点名称，小字显示网址
- **登录方式**：密码/CA/密码+CA
- **账号数**：显示该站点下的账号总数
- **UK数**：显示关联的UK数量，括号显示在库数量（如 `2 (1)`）
- **状态标签**：
  - 🟢 正常：账号可登录 + UK有效
  - 🟡 风险：即将过期/绑定个人手机
  - 🔴 异常：无法登录/UK遗失/账号注销

### 2.6 创建项目时的自动检查

**触发时机**：当用户在"创建项目"页面输入/选择招标平台后

**交互流程**：
```
创建项目表单
├── 项目名称：[________________]
├── 招标平台：[中国政府采购网 ▼]  ← 选择后触发检查
│                               ↓
└──────────────────────────────────────────────────────────┐
│  ⚡ 投标资产检查                                          │
│  ┌──────────────────────────────────────────────────────┐│
│  │  中国政府采购网                      🟢 可投标        ││
│  │  ✅ 账号：已注册  ✅ UK：法人CA在库                  ││
│  │  👤 责任人：张三 (138****1234)                       ││
│  │                                                      ││
│  │  [继续创建]  [查看详情]  [关闭]                      ││
│  └──────────────────────────────────────────────────────┘│
└──────────────────────────────────────────────────────────┘
```

**检查结果的处理**：
- 🟢 可投标：直接提示可继续创建
- 🟡 有风险：显示风险项，用户确认后可继续
- 🔴 不可投标：阻止创建，引导用户先处理资产问题

### 4.1 组件结构
```
src/views/Resource/
├── BAR/                          # 新建BAR模块目录

│   ├── SiteList.vue              # 站点台账列表

│       ├── AssetCard.vue         # 资产状态卡片组件
│       ├── BorrowDialog.vue      # UK借用弹窗
