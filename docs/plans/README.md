# 开发计划 (plans/)

存放近期开发计划与设计稿。

## 目录职责

本目录存放正在执行或即将执行の開発计划、设计稿、实施方案等活跃文档。这些文档是当前开发工作的参考依据。

## 文件命名规范

```
YYYY-MM-DD-<简短描述>-<类型>.md
```

类型后缀：
- `-design.md`: 设计稿
- `-plan.md`: 实施计划
- `-note.md`: 说明文档

## 内容说明

### 保留的近期文档 (2026-04 之后)

- 任务看板定制相关设计
- 任务编辑持久化设计
- 任务状态字典管理设计
- 费用台账相关计划
- 非集成缺口关闭计划

### 已归档文档 (2026-03 及之前)

移至 `../archives/`

## 更新规则

- 计划完成后更新执行状态
- 已完成或废弃的计划移至 `../archives/`
- 保持文档与实际执行情况同步
