## 定位
标讯中心是投标管理系统的**数据入口模块**，负责标讯信息的采集、管理和初步筛选，为后续的投标项目立项提供数据基础。

## 核心目标
- 实现标讯多渠道采集（第三方平台、人工录入、批量导入）
- 实现标讯的智能去重和自动分配
- 实现标讯评估和投标决策的闭环流程
- 实现权限隔离，确保数据安全
- 所有操作留痕，支持审计追溯

## 业务范围
- 标讯的自动采集与人工录入
- 标讯的分配、转派和跟踪
- 标讯评估表的填写和审核
- 标讯投标/放弃决策
- 标讯台账管理和导出
- 标讯源配置管理

## 术语定义
| 术语 | 定义 |
|---|---|
| 标讯 | 招标公告/投标通知的简称，包含招标项目的基本信息 |
| 项目负责人 | 负责跟进标讯评估和项目立项的人员，属于业务/销售侧角色 |

| 业主单位 | 指标讯中的业主单位/采购方单位/甲方 |
| 去重 | 根据特定字段组合判断标讯是否已存在 |

## 4.2.1 角色与权限体系

### 角色定义
四类角色，数据来源统一认证平台：

| 角色 | 说明 |
|---|---|
| 投标管理员 | 投标部门最高管理权限 |
| 投标组长 | 投标小组负责人 |
| 项目负责人 | 业务/销售侧负责标讯跟进 |
| 投标专员 | 投标部门执行人员 |

### 数据权限
| 角色 | 可见范围 |
|---|---|
| 投标管理员 | 全部标讯数据（全局） |
| 投标组长 | 当前：全部；后期规划：仅本组 |
| 项目负责人 | 仅自己创建的 + 分配给自己的 |
| 投标专员 | 仅被分配给自己的 |

### 功能权限矩阵
**列表操作**：管理员/组长有编辑+删除+分发+源配置权限；项目负责人仅限自己的，按状态限制；专员只有查看+搜索+导出。

**数据来源操作**：录入权限普遍，AI解析专员除外，CRM接口创建不可在投标端修改。

**项目负责人编辑权限（按状态）**：待分配不可见；跟踪中可编辑+删除；已评估可编辑评估表（不可删）；后续状态只读。

## 4.2.2 标讯状态与流转

### 状态枚举（7种）
| 状态 | 标签色 | 说明 |
|---|---|---|
| 待分配 | 灰色 | 新拉取/录入，未分配 |
| 跟踪中 | 蓝色 | 已分配，未提交评估 |
| 已评估 | 橙色 | 已提交评估，等待决策 |
| 投标中 | 橙色 | 已确认投标 |
| 已中标 | 绿色 | 项目模块回写 |
| 未中标 | 红色 | 项目模块回写 |
| 已放弃 | 灰色 | 放弃投标 |

### 流转规则（7条路径）
| 流转 | 触发 | 操作人 |
|---|---|---|
| 待分配 → 跟踪中 | 自动/手动分配 | 系统/管理员/组长 |
| 跟踪中 → 已评估 | 提交评估表 | 项目负责人 |
| 已评估 → 投标中 | 立即投标 | 管理员/组长/负责人 |
| 已评估 → 已放弃 | 放弃投标 | 管理员/组长/负责人 |
| 投标中 → 已中标 | 项目结果确认 | 项目模块回写 |
| 投标中 → 未中标 | 项目结果确认 | 项目模块回写 |
| 跟踪中 → 跟踪中(转派) | 转派 | 管理员/组长 |

## 4.2.3 标讯创建

### 四种来源
1. **第三方平台自动拉取**（管理员配置源+定时轮询/一键获取）
2. **CRM商机转入**（CRM确认立项 → 接口创建，字段不可在投标端修改，支持覆盖更新）
3. **人工单条录入**（分步表单：基本信息→评估表，支持AI解析填入）
4. **批量Excel导入**（下载模板→填写→上传）

### 去重规则
项目名称 + 业主单位 + 报名截止时间 + 开标时间 四字段组合完全匹配。

## 4.2.6 流程与操作说明

### 评估提交流程
项目负责人填写评估表 → 选择是否投标 → 提交 → 状态→已评估 → 通知管理员/组长/投标负责人

### 决策操作
> **图片7**（prd-img-AjRtbz62.png，104KB）: 详情页底部【立即投标】【放弃投标】按钮。

- 立即投标 → 创建项目 → 状态→投标中
- 放弃投标 → 填写原因 → 状态→已放弃

### 分发
**自动**：创建时匹配CRM → 有负责人自动分配；无→待分配。
**手动**：管理员/组长 → 分配弹窗（项目负责人必填+投标负责人选填）。
> **图片8**（prd-img-J08DbimR.png，113KB）: 分配弹窗。

### 转派
> **图片9**（prd-img-Rswnbcbh.png，91KB）: 转派弹窗。

可转派状态：跟踪中/已评估。执行人：管理员/组长。转派后原负责人立即失去访问权限。

## 4.2.7 异常处理

| 场景 | 处理 |
|---|---|
| CRM接口失败 | 标讯正常创建（待分配），记异常日志 |
| 去重重复 | 自动跳过/弹窗提示 |
| 评估表提交时状态已变 | 提示刷新 |
| 并发编辑 | 提示已被修改 |
| 分配时负责人不存在 | 提示不存在/已停用 |
| 转派给当前负责人 | 提示不能转派给当前 |
| 导出数据过量 | 提示缩小范围 |
| 网络超时/无网 | 友好提示 |
