# 发布文档 (release/)

存放发布相关文档，包括验收报告和回滚手册。

## 目录职责

本目录存放发布检查清单、验收报告、回滚手册等与发布相关的正式文档。这些文档记录发布决策和执行结果。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `ACCEPTANCE-2026-05-05.md` | 验收报告 | 2026-05-05 版本验收报告 |
| `LIVE_SERVER_DEPLOYMENT_RUNBOOK.md` | 部署手册 | 客户测试服务器 `172.16.38.78` 的完整发布、验活、清理与回滚流程 |
| `PRODUCTION_RELEASE_PIPELINE.md` | 发布流水线说明 | GitHub Actions 主发布链路、变量与目标机目录约定 |
| `ROLLBACK.md` | 回滚手册 | 应用与数据库回滚手册 |

## 更新规则

- 每次重大发布后添加验收报告
- 回滚手册根据实际情况更新
- 保留历史版本以供审计
