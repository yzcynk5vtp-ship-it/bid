# Commercial Scope Freeze

## Purpose
定义正式商用品范围，避免 `API` 模式继续暴露仅适合 `demo/pilot` 的入口。

## Commercial Whitelist

### 认证与基础导航
- 登录、会话恢复、路由守卫
- 系统设置中的真实 API 说明

### 标讯与项目
- 标讯列表、详情、参与投标
- 从标讯创建项目
- 项目列表、详情、基础任务管理

### Knowledge
- 资质库、案例库、模板库的核心 CRUD

### Dashboard
- 真实聚合统计
- 非伪造的下钻数据

### Resources
- 平台账户查询、借用、删除
- 费用列表、费用申请、审批、退还申请、确认退还
- BAR 站点台账 CRUD
- BAR 证书 CRUD、借出、归还、借用记录

### Collaboration / Documents
- 文档章节树展示
- 协作讨论与变更记录展示
- 版本历史查询、版本对比、版本回滚

## Demo-Only Blacklist

以下能力保留给 `mock` 模式，不属于正式商用品范围：

- 标讯级 AI 分析页面与入口
- 项目创建页中的评分预分析步骤
- 协作中心中的章节负责人分配、提醒、锁定、批量保存
- 版本管理中的“创建新版本”快捷入口
- BAR 站点验证、站点启停、导入
- 平台账户新增表单、账户启停
- 所有仍需页面推导或虚构后端成功态的高级交互

## Gate Rule
- `API` 模式下，白名单功能必须真实闭环。
- `API` 模式下，黑名单功能不得继续作为用户可点击入口出现。
- 新增功能若未进入白名单，不得默认进入正式版导航和主流程。
