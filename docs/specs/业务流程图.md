## 方案一：泳道流程图（Mermaid）

```mermaid
flowchart TD
    %% 定义样式
    classDef startEnd fill:#67C23A,stroke:#333,stroke-width:2px,color:#fff
    classDef process fill:#409EFF,stroke:#333,stroke-width:1px,color:#fff
    classDef decision fill:#E6A23C,stroke:#333,stroke-width:1px,color:#fff
    classDef data fill:#909399,stroke:#333,stroke-width:1px,color:#fff
    classDef ai fill:#F56C6C,stroke:#333,stroke-width:1px,color:#fff

    %% 开始
    Start([开始]):::startEnd

    %% 阶段1：标讯获取
    subgraph Phase1 [第一阶段：标讯获取]
        direction TB
        A1[外部标讯获取]:::process
        A2[第三方商机服务]:::data
        A3[智能分发规则]:::decision
        A4[销售指派跟进]:::process
    end

    %% 阶段2：项目立项
    subgraph Phase2 [第二阶段：项目立项]
        direction TB
        B1[CRM客户同步]:::data
        B2[创建投标项目]:::process
        B3[立项审批流程]:::decision
    end

    %% 阶段3：任务分解
    subgraph Phase3 [第三阶段：任务分解]
        direction TB
        C1[智能任务拆解]:::ai
        C2[4阶段看板]:::process
        C3[待办→进行中→待审核→完成]:::process
        C4[交付物上传]:::data
    end

    %% 阶段4：标书编制
    subgraph Phase4 [第四阶段：标书编制]
        direction TB
        D1[模板智能装配]:::ai
        D2[协同编辑]:::process
        D3[AI合规检查]:::ai
        D4[AI质量检查]:::ai
    end

    %% 阶段5：投标提交
    subgraph Phase5 [第五阶段：投标提交]
        direction TB
        E1[用印申请]:::process
        E2[封装提交]:::process
        E3[状态跟踪]:::data
    end

    %% 阶段6：结果闭环
    subgraph Phase6 [第六阶段：结果闭环]
        direction TB
        F1[中标登记]:::process
        F2[结果分析]:::data
        F3[案例沉淀]:::data
        F4[竞对信息]:::data
    end

    %% 结束
    End([结束]):::startEnd

    %% 连接关系
    Start --> A1
    A1 --> A2
    A1 --> A3
    A3 --> A4
    A4 --> B1
    B1 --> B2
    B2 --> B3
    B3 --> C1
    C1 --> C2
    C2 --> C3
    C3 --> C4
    C4 --> D1
    D1 --> D2
    D2 --> D3
    D2 --> D4
    D3 --> E1
    D4 --> E1
    E1 --> E2
    E2 --> E3
    E3 --> F1
    F1 --> F2
    F2 --> F3
    F2 --> F4
    F3 --> End
    F4 --> End
```

## 使用说明

### Mermaid图表使用方法：
1. **Markdown编辑器**：直接复制代码块到支持Mermaid的编辑器（如Typora、Obsidian、VS Code）
2. **在线渲染**：访问 https://mermaid.live 粘贴代码渲染
3. **导出图片**：渲染后可导出为PNG/SVG格式
4. **PPT使用**：导出PNG后插入PPT

### 推荐方案：
- **讲标演示**：方案三（简化流程图）或方案六（ASCII版）
- **技术文档**：方案一（泳道流程图）或方案二（水平流程图）
- **详细说明**：方案五（带角色泳道）
