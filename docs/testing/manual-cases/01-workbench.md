# 工作台 — 人工测试用例

> 蓝图章节：§4.1
> 测试角色：admin / manager / staff

## TC-WB-001：工作台首页加载

**前置条件**：登录系统

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 使用 admin 登录 | 进入工作台 |
| 2 | 查看统计卡片 | 显示标讯/项目/待办/公告数量 |
| 3 | 查看快捷入口 | 显示"快速创建标讯""创建项目"等入口 |
| 4 | 查看待办事项 | 显示当前用户的待办任务列表 |

## TC-WB-002：快捷入口导航

**前置条件**：登录系统

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 点击"快速创建标讯" | 跳转到标讯创建页 |
| 2 | 点击"创建项目" | 跳转到项目创建页 |
| 3 | 点击"我的任务" | 跳转到任务列表 |
| 4 | 点击"日程" | 跳转到日程视图 |
