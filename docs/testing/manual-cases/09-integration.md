# 系统集成 — 人工测试用例

> 蓝图章节：系统集成
> 测试角色：admin

## TC-IN-001：系统集成面板

**前置条件**：admin 登录

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 打开系统设置 → 系统集成 | 显示 4 张集成卡片 |
| 2 | 查看企业微信卡片 | 显示 Corp ID、Agent ID 等字段 |
| 3 | 查看泛微 OA 卡片 | 显示 OA 地址、账号密钥字段 |
| 4 | 查看 CRM 卡片 | 显示 CRM 地址、Token、Client ID |
| 5 | 查看组织架构同步 | 显示 SDK 配置字段 |

## TC-IN-002：CRM 配置

**前置条件**：admin 登录

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 填写 CRM 地址 | 输入可编辑 |
| 2 | 填写 Token | 密码模式显示 |
| 3 | 点击"测试连接" | 显示连接结果 |
| 4 | 点击"保存配置" | 配置保存成功 |
