## 蓝图功能点（从 PRD / gap-analysis / 历史实现笔记提取）

- 标讯创建后，未匹配 CRM 项目负责人 → 状态保持 PENDING_ASSIGNMENT。
- PENDING_ASSIGNMENT 标讯详情页必须展示【分配】按钮（bid_admin / bid_lead / admin 可见）。
- 销售等非管理角色不可见分配按钮。
- 点击分配 → 打开弹窗 → 选择项目负责人 → 提交 → 状态推进为 TRACKING。
- 分配后详情页按钮切换为【转派】。
