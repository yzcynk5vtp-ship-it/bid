# 项目经理个人分享 PPT 叙事计划

## 受众与目标

受众为项目启动会参会人员、客户关键用户、甲乙方项目组。目标是在正式 SOW 汇报之外，以项目经理个人视角分享一套更适合 AI 时代的项目协作方法，并清晰说明“客户商机中心”作为赠送试点能力的价值与边界。

## 叙事主线

1. 项目最难的不是技术，而是需求共识、确认与变更拉扯。
2. 用会议留痕、AI 第三视角、原型逼近法降低沟通成本。
3. 用 Agent 与技能复制项目经理的工作方法。
4. 客户商机中心可以作为赠送试点能力，帮助客户从“看标讯”走向“经营客户”，但不纳入验收标准。

## 幻灯片结构

1. 封面
2. 项目最难的是共识成本
3. 会议留痕：录音、转写、纪要、待办
4. 分歧处理：先结构化，再请 AI 做第三视角
5. 需求调研：坚持原型逼近法
6. 用 Agent 工作，用技能复制自己
7. 赠送试点：客户商机中心
8. 边界说清：可试用、不验收、不承诺预测效果
9. 共识收束：把会议变资产，把经验变能力

## 视觉系统

蓝色主基调，白底商务风；标题用深蓝，重点用亮蓝，风险边界用浅蓝灰承载。采用大标题、少量短句、三卡片和流程箭头，保证简单易懂。
