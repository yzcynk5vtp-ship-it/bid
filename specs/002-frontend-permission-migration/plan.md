## Summary

Migrate the Vue 3 frontend from hardcoded Legacy role checks (`role === 'admin'`) to `menuPermissions`-driven access control. The backend already supports 10 `RoleProfile` codes and returns `menuPermissions` in `AuthResponse`. The frontend must consume this array for route guards, sidebar filtering, and page-level feature visibility.

Delivery is phased: P1 unblocks new roles from route-guard denial; P2 cleans page-level hardcoded checks; P3 removes deprecated `meta.roles` entirely.
