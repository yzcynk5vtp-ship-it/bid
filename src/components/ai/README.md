# AI Components 模块

> 一旦我所属的文件夹有所变化，请更新我。

## 职责说明
该目录承载 AI 相关的可复用组件，覆盖分析、协作、版本、任务和配置等面板。
这里的组件面向多个页面复用，不放页面路由或业务编排逻辑。
组件职责以展示和局部交互为主，复杂状态由页面层接管。
