# Layout 模块

> 一旦我所属的文件夹有所变化，请更新我。

## 职责说明
布局模块负责应用级外壳结构，包括顶部栏、侧边栏和主容器。
侧边栏采用白色底色承载导航层级，并以 DESIGN.md 中的西域 Logo Green 作为活动态、品牌字标和菜单强调色，保持轻量清晰的工作台视觉体系。
顶部栏承载通知、用户下拉和低频管理入口；操作日志作为用户下拉中的快捷入口跳转到系统设置的审计日志面板，不占用左侧一级导航。
该目录不放业务页面逻辑，只承载导航、容器和全局布局编排。
业务页面通过这些布局组件获得统一的框架和导航体验。
