# Resource 模块

> 一旦我所属的文件夹有所变化，请更新我。

## 职责说明
资源模块负责账户、费用、投标结果等资源型业务页面。
该目录聚焦资源申请、查询、流转和归档，不承载基础组件实现。
子目录按资源类型继续拆分，保持页面边界清晰。
费用页现已拆分为容器页 + `expense/` 子模块，并接通保证金预计退还日期、自动跟踪状态和手工退还提醒的真实 API 链路。
合同借阅页接入独立分页 API、生命周期动作和本地错误提示。

## 边界清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `Account.vue` | View | 平台账户页 |
| `BidResult.vue` | View | 投标结果页 |
| `ContractBorrow.vue` | View | 合同借阅申请、审批、归还和事件历史页 |
| `contractBorrow.css` | Style | 合同借阅页局部样式 |
| `Expense.vue` | View | 费用管理页容器 |
| `expense/` | 目录 | 费用页组合式逻辑、保证金跟踪与子组件 |
| `BAR/` | 目录 | BAR 资源子模块 |
